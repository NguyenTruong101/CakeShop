## Trong tương lai:

-	Sử dụng Machine Learning để thống kê

### Hướng dẫn cài đặt thư viện hỗ trợ đề tài (frontend & backend)
<h5>Vào terminal và thực hiện</h5>

```
npm install
cd frontend
npm install
```
### Hướng dẫn chạy chương trình (frontend & backend)
<h5>Vào terminal và thực hiện</h5>

```
# Run frontend (:3000) & backend (:4000)
cd frontend
npm start

# Run backend only
npm run prod
```
